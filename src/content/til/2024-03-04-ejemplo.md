Hoy aprendi a usar `git bisect` para encontrar el commit exacto que introdujo un bug.

## Como funciona

Git bisect usa busqueda binaria para encontrar el commit problematico:

```bash
# Iniciar bisect
git bisect start

# Marcar el commit actual como malo
git bisect bad

# Marcar un commit antiguo que funcionaba
git bisect good abc123
```

Git automaticamente hace checkout de commits intermedios. En cada paso, pruebas si el bug existe y marcas:

```bash
git bisect good  # si funciona
git bisect bad   # si tiene el bug
```

## Automatizarlo

Si tienes un script que detecta el bug:

```bash
git bisect run ./test-bug.sh
```

Git ejecuta el script en cada commit hasta encontrar el culpable.

## Terminar

```bash
git bisect reset
```

Esto te devuelve a tu branch original.

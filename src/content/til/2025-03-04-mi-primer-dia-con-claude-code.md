Hoy he empezado a usar Claude Code por primera vez y he conseguido montar este blog desde cero. Ya había instalado CC pero hoy ha sido el día de ponerme en serio.

## Lo que hice

Abri la terminal y lancé Claude Code. Le pedí que me hiciera un plan para crear un blog minimalista tipo TIL (Today I Learned.

Claude me devolvio un plan con la estructura del proyecto, y los pasos a seguir. Lo lei, entendí ciertas cosas, otras no, y lo que. no entendí se lo escupí a Claude para que me lo explicase. Lo entendi y le di luz verde para implementarlo.


## Configurando Git

Nunca habia usado GitHub desde la terminal, asi que tuve que instalar la CLI de GitHub (`gh`) para poder autenticarme y subir el repositorio.

Una vez configurado, conecte el proyecto a mi cuenta de GitHub y lo subi con un simple push. Primer repositorio creado. La madre que me recontramilparió.

## Editando estilos
Una vez que funcionaba empecé a probar a tocar estilos. Y es facilísimo. Sé un poco de CSS por lo que con el inspector de chrome iba viendo que queria cambiar y le decía a Claude Code las instrucciones exactas. Tipo, oye aquí esta tipografía, aquí este interlineado.... y en segundos estaba.

## Añadiendo entradas
Le pregunté a Claude como hacía una entrada nueva, ya que normalmente en servicios como Wordpress tienes una interfaz. La respuesta? Me dijo: PUedes hacer X,Y,Z (Un follón) o, simplemente: Pídeselo a Claude Code. Y eso hice. 2 segundos después aquí estoy escribiendo la entrada. Jajajajaja increíble.


## Hostearlo en Vercel
Me falta un paso que lo he leído pero no he hecho que ahora tengo que hostear publicamente mi repositorio que está en Github en Vercel, que era lo que me recomendaba Claude Code y es gratis. Porque uno es curioso pero no estamos para tirar el dinero monstro.


## Resultado

Si esto está online es que he conseguido en menos de una hora un blog funcional.
